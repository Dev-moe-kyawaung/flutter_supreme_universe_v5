## 🚀 Quick Start

### **Prerequisites**
```bash
# Install Flutter
curl -fsSL https://flutter.dev/install | bash

# Verify
flutter doctor
```
## Installation
```bash
# Clone repository
git clone https://github.com/Dev-moe-kyawaung/flutter-supreme-universe-V5.git

# Navigate
cd Moekyawaung-portfolios

# Get dependencies
flutter pub get

# Run for web (development)
flutter run -d chrome

# Run for mobile browser
flutter run -d web-server --web-port=8080
```
# Build for All Platforms
```bash
# Web (PWA)
flutter build web --release

# Android APK
flutter build apk --release

# iOS
flutter build ios --release --no-codesign
```
